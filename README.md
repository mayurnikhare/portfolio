# 🌐 Personal Portfolio Website

## 📌 Overview
This is my personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, projects, and contact information in a clean, responsive, and user-friendly design. The goal of this project is to present my work and abilities in a professional and interactive way.

---

## 🚀 Features
- Fully responsive design (mobile + desktop friendly)
- Modern and clean UI/UX design
- Smooth navigation between sections
- About, Skills, Projects, and Contact sections
- Interactive elements using JavaScript
- Resume download option

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
